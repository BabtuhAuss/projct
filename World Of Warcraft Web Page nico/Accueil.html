Ce site vous est proposé par Nicolas Castello et Aussenac Baptiste dans le but d'un projet tuteuré.

Dans ce site vous pourrez voir plusieurs informations concernant l'univers du jeu vidéo: Warcraft, ainsi que de ses suites.

Le site web est représenté sous differentes parties:
  • Dans la première partie, il est question de l'histoire contenu dans le jeu, séparé en 3 grandes guerres qui ont boulversé le monde.
  • Dans la deuxième partie, les factions sont présentés et des héros de chaques faction ont une petite biographie.
  • Dans la troisième partie, il y a plusieurs cartes qui montrent les différents lieux de Warcraft.
  • Pour finir, une page Support est présente dans le but de recevoir les avis des visiteurs qui observent le site (comme il est notion de PHP dans cette partie nous avons fait comme si nous connaissions les bases mais tout envoi d'information sera refusé)

Nous vous prions de passer un bon moment sur ce site !

Si vous voulez vous aventurer sur un Quizz que nous avons elaboré rien que pour vous, revenez une fois que vous avez lu toute les pages, et ensuite cliquez sur cette IMAGE !
